API

      jQuery([selector]}).autoscroll(options);

Options

scrollTime
the amount of time taken to scroll the focussed element to the center
delay
the amount of time the script waits before starting to scroll
horizontal
bool, true if the elements should be scrolled to center horizontally
vertical
bool, true if the elements should be scrolled to center vertically

{default values}
$.fn.fscroll.defaults = {
  scrollTime: 1000,
  delay: 500,
  horizontal: false,
  vertical: true
};

Usage

       // scrolls all input elements within tables within a second of being focussed.
       $(function(){
        $('table > input').fscroll({ scrollTime: 1000 });
      });
